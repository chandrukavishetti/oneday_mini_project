================================================================================
                    DATABASE SETUP DETAILS
================================================================================

Database Name: expense_claim_db
Database Type: MySQL

Create Database:
CREATE DATABASE IF NOT EXISTS expense_claim_db;

Tables Created:
1. claims - Stores all expense claims
   - id (BIGINT, PRIMARY KEY, AUTO_INCREMENT)
   - employee_name (VARCHAR, NOT NULL)
   - department (ENUM, NOT NULL)
   - category (ENUM, NOT NULL)
   - amount (DECIMAL(15,2), NOT NULL)
   - expense_date (DATE, NOT NULL)
   - description (VARCHAR(500), NOT NULL)
   - status (ENUM, NOT NULL, DEFAULT 'PENDING')
   - review_remark (VARCHAR(500))
   - month_year (VARCHAR(7), NOT NULL)
   - created_at (TIMESTAMP)
   - updated_at (TIMESTAMP)

2. department_budgets - Stores monthly department budgets
   - id (BIGINT, PRIMARY KEY, AUTO_INCREMENT)
   - department (ENUM, NOT NULL)
   - month (INT, NOT NULL)
   - year (INT, NOT NULL)
   - budget_amount (DECIMAL(15,2), NOT NULL)
   - created_at (TIMESTAMP)
   - updated_at (TIMESTAMP)
   - UNIQUE KEY (department, month, year)

================================================================================
                    API ENDPOINTS SUMMARY
================================================================================

Claims Endpoints:
POST   /api/claims/submit          - Submit a new claim
PUT    /api/claims/review          - Review a claim (approve/reject)
GET    /api/claims/all             - Get all claims
GET    /api/claims/{id}            - Get claim by ID
GET    /api/claims/department/{dept} - Get claims by department
GET    /api/claims/status/{status} - Get claims by status
GET    /api/claims/month/{monthYear} - Get claims by month
GET    /api/claims/filter          - Get claims with filters
GET    /api/claims/summary         - Get monthly summary

Budget Endpoints:
POST   /api/budgets/create         - Create a new budget
PUT    /api/budgets/update/{id}    - Update a budget
GET    /api/budgets/all            - Get all budgets
GET    /api/budgets/{id}           - Get budget by ID
GET    /api/budgets/find           - Find budget by dept/month/year
DELETE /api/budgets/{id}           - Delete a budget
GET    /api/budgets/exists         - Check if budget exists

================================================================================
                    RUNNING THE APPLICATION
================================================================================

1. Start MySQL Database:
   mysql -u root -p
   CREATE DATABASE IF NOT EXISTS expense_claim_db;

2. Run Backend:
   cd expense-claim-backend
   mvn clean install
   mvn spring-boot:run
   Backend runs on: http://localhost:8080

3. Run Frontend:
   cd expense-claim-frontend
   npm install
   npm run dev
   Frontend runs on: http://localhost:5173

4. Access the Application:
